### Exercise 1: The Password Matcher
**Problem:** Write a function that compares two integer arrays (an `input` code vs. a `correct` code).
* Return `1` (True) if **every** number matches.
* Return `0` (False) if **any** number is different.

**Example:**
* Input: `input = [1, 2, 3]`, `correct = [1, 2, 3]` -> **Returns 1**
* Input: `input = [1, 2, 3]`, `correct = [1, 2, 5]` -> **Returns 0**
