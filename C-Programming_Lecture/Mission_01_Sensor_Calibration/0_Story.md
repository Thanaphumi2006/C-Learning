Mission 01: Establish Link (System Boot)

The Story: The Mars Rover "Pathfinder" has just landed on the red planet. It is currently in "Sleep Mode." You are the engineer at Mission Control who must remotely boot up its Operating System.

The Objective:

Initialize the communication uplink (printf).

Check memory status (int, float, char).

Calculate the weight of the initial supplies found in the cargo bay (Scanf & Math).

Coding Skills: printf, Variables, Arithmetic, Input.
